# 📖 Overview

WAVE is a web-based motorbike wash management system designed to improve customer experience and optimize daily operations.

The system provides:

- Online booking
- Tier-based loyalty program
- Customer and vehicle management
- Promotion management
- Wash history tracking

The platform helps businesses increase customer retention through personalized rewards and priority booking features.

---

# ✨ Main Features

## 👤 Customer Features

- Online booking
- Loyalty points tracking
- Wash history management
- Vehicle management
- Tier-based booking priority

| Tier     | Booking Window |

# 🚀 Getting Started

## Prerequisites

- Node.js >= 18
- pnpm

Install pnpm globally:

```bash
npm install -g pnpm
```

# 📥 Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project folder:

```bash
cd <project-folder>
```

Install dependencies:

```bash
pnpm install
```

---

# ▶️ Run

Start the project:

```bash
pnpm dev
```

Open your browser at:

```bash
http://localhost:3000
```

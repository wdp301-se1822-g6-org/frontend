# Frontend Technologies & Libraries

Bảng danh sách các công nghệ, thư viện chính được sử dụng ở phía Frontend của dự án **Wash Auto** kèm link tài liệu chính thức và vai trò thực tế:

| Công nghệ / Thư viện | Link tài liệu chính thức | Vai trò / Công dụng trong dự án |
| :--- | :--- | :--- |
| **Next.js 16** | [https://nextjs.org](https://nextjs.org) | Khung sườn chính (React Framework) quản lý routing (App Router), rendering và tối ưu hóa hệ thống trang. |
| **React 19** | [https://react.dev](https://react.dev) | Thư viện UI cốt lõi để xây dựng các Functional Components, sử dụng Hooks điều phối logic giao diện. |
| **Tailwind CSS v4** | [https://tailwindcss.com](https://tailwindcss.com) | Framework CSS tối ưu hóa giao diện nhanh chóng bằng utility classes, chịu trách nhiệm styling responsive, dark mode. |
| **PostCSS** | [https://postcss.org](https://postcss.org) | Công cụ tiền xử lý CSS, biên dịch và tối ưu hóa file CSS (tích hợp trực tiếp cùng Tailwind v4). |
| **Axios** | [https://axios-http.com](https://axios-http.com) | Thư viện HTTP client gửi request/response lên REST API Backend, cấu hình interceptor tự động refresh token. |
| **React Query (TanStack)** | [https://tanstack.com/query/latest](https://tanstack.com/query/latest) | Quản lý state server, lưu cache dữ liệu API đơn hàng, ca trực, thợ rửa và tự động đồng bộ/làm mới (invalidateQueries). |
| **Zustand** | [https://zustand.docs.pmnd.rs](https://zustand.docs.pmnd.rs) | Thư viện quản lý state toàn cục siêu nhẹ, dùng để lưu trạng thái đăng nhập (Authentication Store) của người dùng. |
| **Sonner (Toaster)** | [https://sonner.emilkowal.ski](https://sonner.emilkowal.ski) | Thư viện quản lý thông báo Toast thông minh, có tích hợp close button để cải thiện trải nghiệm đóng nhanh của người dùng. |
| **Zod** | [https://zod.dev](https://zod.dev) | Thư viện kiểm chứng dữ liệu (Data validation) ở phía client, bảo đảm an toàn dữ liệu trước khi gửi lên Backend. |
| **React Hook Form** | [https://react-hook-form.com](https://react-hook-form.com) | Quản lý hiệu năng biểu mẫu (Form handling) và kiểm soát trạng thái đầu vào của input tối ưu. |
| **Lucide React** | [https://lucide.dev](https://lucide.dev) | Bộ sưu tập icon vector dạng SVG sắc nét, đồng bộ hóa phong cách icon trên toàn bộ giao diện quản trị & khách hàng. |
| **TypeScript** | [https://www.typescriptlang.org](https://www.typescriptlang.org) | Ngôn ngữ gõ tĩnh (Static typing) giúp phát hiện lỗi code ngay khi biên dịch, bảo toàn kiểu dữ liệu cho toàn bộ API DTOs. |
| **Eslint** | [https://eslint.org](https://eslint.org) | Công cụ phân tích mã nguồn tĩnh để phát hiện nhanh các cú pháp lỗi, định hình style viết code nhất quán. |
| **PWA (Service Worker)** | [https://web.dev/progressive-web-apps](https://web.dev/progressive-web-apps) | Cấu hình cài đặt ứng dụng tiến trình (Manifest & Service Worker) để hỗ trợ cài đặt app lên điện thoại/máy tính giống app native. |
